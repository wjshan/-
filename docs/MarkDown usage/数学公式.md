# 数学公式

## 行内公式

将公式插入行内

```
$公式内容$
```

如:$xyz$

## 单行公式

将公式作为单独一行,并且居中.最好在 `$$`处换行

```
$$
公式内容
$$
```

如:

$$
 \sigma(z) = \int_0^\infty t^{z-1}e^{-t}dt\,
$$
